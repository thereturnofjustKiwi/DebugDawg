Upload Network Data File

Choose a CSV or JSON file containing your network flow data.

Example: sample_network_data.csv

Process and Segregate Data

Click "Upload and Process" to parse the file and separate benign from intrusion data.

Output: Displays first 10 rows of segregated data
3

Generate Data Reports

Click "Generate Report" to see visual charts for traffic distribution.

Example Chart: Pie chart showing 'Benign' vs 'Intrusion'
4

Verify Blockchain Integrity

Check the threat log chain hashes for data integrity validation.

Verified chain shows "Yes" for valid entries
5

Run Real-Time Simulation

Start a live simulation of network flows and watch detection results.

Alerts will be highlighted as 'Intrusion' and actions shown
6

Enter flow features to get trusted automated explanation of predictions.

Example input: 0,1,45,9,0,... (41 features)
